Module TicTacToe¶

tictactoe.actions(board)¶

Retorna el conjunto de todas las acciones posibles (i, j) disponibles en el tablero.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

Un conjunto de tuplas (i, j) que representan las posiciones vacías.

Tipo del valor devuelto:

set

tictactoe.basic_minimax(board)¶

Algoritmo Minimax básico sin Poda Alfa-Beta.

Explora todo el árbol de juego sin poda.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

Un tuple con el valor de utilidad y la mejor acción.

Tipo del valor devuelto:

tuple

tictactoe.get_columns(board)¶

Retorna las columnas del tablero.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

Una lista con las tres columnas del tablero.

Tipo del valor devuelto:

list

tictactoe.get_diagonal(board)¶

Retorna las diagonales del tablero.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

Una lista con las dos diagonales del tablero.

Tipo del valor devuelto:

list

tictactoe.initial_state()¶

Retorna el estado inicial del tablero.

Devuelve:

Una lista de listas que representa el tablero vacío.

Tipo del valor devuelto:

list

tictactoe.max_alpha_beta_pruning(board, alpha, beta)¶

Realiza la poda Alfa-Beta para el jugador X (Maximiza).

Parámetros:

  • board (list) – El tablero de juego.

  • alpha (float) – El valor alfa para la poda.

  • beta (float) – El valor beta para la poda.

Devuelve:

Un tuple con el valor de utilidad y la mejor acción.

Tipo del valor devuelto:

tuple

tictactoe.min_alpha_beta_pruning(board, alpha, beta)¶

Realiza la poda Alfa-Beta para el jugador O (Minimiza).

Parámetros:

  • board (list) – El tablero de juego.

  • alpha (float) – El valor alfa para la poda.

  • beta (float) – El valor beta para la poda.

Devuelve:

Un tuple con el valor de utilidad y la mejor acción.

Tipo del valor devuelto:

tuple

tictactoe.minimax(board)¶

Retorna la acción óptima para el jugador actual en el tablero.

Utiliza Poda Alfa-Beta para optimización.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

La mejor acción posible para el jugador actual.

Tipo del valor devuelto:

tuple or None

tictactoe.player(board)¶

Retorna el jugador que tiene el siguiente turno en el tablero.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

“X” si es el turno de X, “O” si es el turno de O.

Tipo del valor devuelto:

str

tictactoe.result(board, action)¶

Retorna el tablero que resulta al hacer el movimiento (i, j) en el tablero.

Parámetros:

  • board (list) – El tablero de juego.

  • action (tuple) – La acción a realizar, una tupla (i, j).

Muestra:

Exception – Si la posición (i, j) ya está ocupada.

Devuelve:

Un nuevo tablero con el movimiento realizado.

Tipo del valor devuelto:

list

tictactoe.terminal(board)¶

Retorna True si el juego ha terminado (ganador o empate), False en caso contrario.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

True si hay un ganador o si todas las casillas están llenas, False en caso contrario.

Tipo del valor devuelto:

bool

tictactoe.three_in_a_row(row)¶

Verifica si una fila tiene tres elementos iguales.

Parámetros:

row (list) – Una fila del tablero.

Devuelve:

True si todos los elementos de la fila son iguales y no son None, False en caso contrario.

Tipo del valor devuelto:

bool

tictactoe.utility(board)¶

Retorna 1 si X ha ganado, -1 si O ha ganado, 0 en caso contrario.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

1 si X ha ganado, -1 si O ha ganado, 0 en caso de empate o si el juego no ha terminado.

Tipo del valor devuelto:

int

tictactoe.winner(board)¶

Retorna el ganador del juego, si existe.

Verifica todas las filas, columnas y diagonales.

Parámetros:

board (list) – El tablero de juego.

Devuelve:

“X” si el jugador X ha ganado, “O” si el jugador O ha ganado, None si no hay ganador.

Tipo del valor devuelto:

str or None
